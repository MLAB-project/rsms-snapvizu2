# snapvizu2

To run:

	python3 server.py

Server listens on port 81. One can override the default snapshot directory (`/tmp/snaps`) through the `SNAPVIZU2_SNAP_DIR` environment variable, e.g.

	SNAPVIZU2_SNAP_DIR=/var/snaps/ python3 server.py
